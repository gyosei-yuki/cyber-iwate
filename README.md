## データ出典・クレジット
本プロジェクトの3D空間および人口データは、以下のオープンデータを基に、渋谷佑生（行政書士 渋谷佑生 事務所）がGeoJSONへの変換・3D化等の加工を行って作成したものです。

1. 岩手県オープンデータポータル「DataEYE」 本データは「CC BY（表示）」に基づき、一部加工して利用しています。 出典：岩手県及び市町村共同オープンデータポータルサイト「DataEYE」（https://dataeye.iwate.jp/）

2. 政府統計の総合窓口（e-Stat） 出典：「政府統計の総合窓口（e-Stat）」（https://www.e-stat.go.jp/） ※本データは加工して利用しています。
