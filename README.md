## データ出典・クレジット
本プロジェクトの3D空間および人口データは、以下のオープンデータを基に、渋谷佑生（行政書士 渋谷佑生 事務所）がGeoJSONへの変換・3D化等の加工を行って作成したものです。

* 出典1：「盛岡市オープンデータ（人口と世帯数）」（盛岡市）
* 出典2：「〇〇調査（境界線データなど）」（総務省統計局 / e-Stat）を加工して作成
